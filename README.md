# transwarp-nlp

深度自然语言处理工具集

## 功能实现

- 中文分词
- 中文序列化标注
- 中文命名实体识别
- 中文关键词抽取（实现中）
- 中文文本自动摘要
- 依存句法分析
- 中文文本分类（实现中）
- 中文自由写诗（实现中）
- 中文对话系统（实现中）
- 中文问答系统（实现中）
- 中英机器翻译（实现中）
- 语音识别（实现中）

## 依赖

* tensorflow (>= r1.0)
* numpy
* pygtrie
